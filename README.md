# Pranav Kanungo - Portfolio

A premium, interactive personal portfolio showcasing my skills, projects, and professional journey as a Full Stack Developer.

## 🚀 Features

- **Interactive UI/UX**: Cinematic animations and glassmorphism design.
- **Dynamic Projects**: Detailed cards for featured projects with live demo and GitHub links.
- **Responsive Design**: Optimized for all devices, from mobile to desktop.
- **Skill Arsenal**: A showcase of technologies I specialize in (Frontend, Backend, and Tools).
- **Contact Form**: Integrated contact section for professional inquiries.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Custom styling with HSL colors and glassmorphism.
- **JavaScript**: Interactive elements and scroll animations.
- **Google Fonts**: Modern typography (Outfit).

## 📂 Project Structure

- `index.html`: Main structure of the portfolio.
- `style.css`: Custom design tokens, layout, and animations.
- `script.js`: Interactive logic and reveal-on-scroll animations.
